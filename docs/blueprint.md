# **App Name**: Client Tracker

## Core Features:

- Client Input: Input form for adding client details (name, project, payment, date).
- Client Record Display: Display of client records in a table or list format.
- Sorting: Sort clients by name, project, payment amount, or date.

## Style Guidelines:

- Primary color: Light gray or white for the background.
- Secondary color: Dark blue or gray for text and headers.
- Accent: Teal (#008080) for interactive elements and highlights.
- Clean and simple layout with clear sections for input and display.
- Simple, professional icons for sorting and actions.
